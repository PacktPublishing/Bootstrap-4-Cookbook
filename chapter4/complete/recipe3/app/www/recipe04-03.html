Using tether options in JS to alter the behavior of popups

In this simple recipe, we will utilize the options provided by the tether library to alter the behavior of our popup.

The options we will alter will involve both using the html data-* attributes, as well as the JavaScript options as described in tether documentation, namely constraint options, and offset options.

Looking at the html attributes, we can see that the options we can use include the title attribute, which gives card-like styling to the popover's heading (title).

Looking at the JS options, we can inspect the code called on document ready:


	$(function () {
	  $('[data-toggle="popover"]').popover({
		trigger: 'hover', // trigger: 'focus',
		delay: { "show": 500, "hide": 100 },
		attachment: 'top left',
		targetAttachment: 'top right',
		constraints: [
			{
				to: 'window',
				pin: true
			}
		] 	// constraints
	  }); // popover
	}); // function

The above code is pretty easy to ubderstand. Among other things, we can set the trigger for the popover, as well as the delay to show it and hide it based on the triggerable events.
