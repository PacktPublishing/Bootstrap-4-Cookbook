Recipe 05-03: Centering navbar links

Link to bootstrap navbar docs

In this recipe, we continue using a customized version of the navbar used on the official bootstrap 4 website. However, in each recipe, we are also tweaking the navbar slightly, thereby making it easier to notice the differences between each recipe while still working with the exact same links throughout this chapter.

To achieve that, we are setting the flex property to the value of auto, in the custom class of flex-links. This class is used on the ul element inside the div with the class of .navbar-collapse.collapse.
