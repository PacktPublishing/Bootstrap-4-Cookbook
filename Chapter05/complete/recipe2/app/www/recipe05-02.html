Recipe 05-02: Placing a single Bootstrap navbar dropdown to the right

Link to bootstrap navbar docs

We are using a customized version of the navbar used on the official bootstrap 4 website.

The navbar is somewhat similar to the one used in the previous recipe.

To achieve the navbar as it is in this recipe, we are using the usual div with classes .navbar-collapse.collapse.

Inside of that div, we are using two ul elements. The first ul element is styled using the classes of .nav.navbar-nav and .mr-auto.

The third class, .mr-auto, is the key here. This class will increase the right margin on the ul element to stretch so that it pushes the next ul all the way to the right edge of the navbar.

This raises another issue, which is, since the second ul element holds the dropdown items, clicking the "Recipes" dropdown will display the items in the dropdown with the right-side cut-off.

Therefore, we need to re-align the dropdown bubble, and we achieve it by using the class of .dropdown-menu-right.
