Recipe 09-01: Customizing Bootstrap Builds by Cherry-picking Sass Partials

Link to bootstrap docs

This web page is using a CSS stylesheet compiled from only a few scss partial files. Many files were not included, such as partials for components such as the jumbotron, alert, progress etc.

For example, the code for the button below was not included, and thus the regular Bootstrap 4 styles for buttons no longer apply.

To make the button work again, uncomment the appropriate scss partial, as explained in step 11 of recipe 09-01.
